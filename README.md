# IPL-2022
IPL Data Analysis &amp; Visualization project using Python, Pandas, and Seaborn to uncover insights from match data.
# 🏏 IPL Data Analysis Project

This project performs Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset to uncover meaningful insights about teams, players, and match performances.

## 📌 About IPL

The Indian Premier League (IPL) is one of the most popular T20 cricket leagues in the world, featuring top international and domestic players.

## 🚀 Features

* Data cleaning and preprocessing
* Analysis of team performance
* Player statistics (top scorers, wicket-takers)
* Match-winning trends
* Beautiful data visualizations

## 📊 Key Insights

* Most successful teams in IPL history
* Top run scorers and wicket takers
* Toss impact on match results
* Venue-based performance analysis

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📂 Dataset

* IPL match dataset (CSV format)
* Includes match results, players, scores, venues

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/ipl-analysis.git
```

2. Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Run Jupyter Notebook:

```bash
jupyter notebook
```
